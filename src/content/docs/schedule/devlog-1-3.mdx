### DevLog 1-3 

#### Create a Node.js project and publish it on Github

This [Introduction to NPM and Node](https://omundy.github.io/learn-javascript/topics/node/slides.html) tutorial builds a foundation for what you will learn in this class. Below find the learning objectives for this DevLog which correspond to sections in the tutorial. Save the information with the ✏️ which you will enter in your DevLog in the next class.

Install and use Node.js on your machine
Confirm your Node version ✏️v25.4.0
Run a script with Node.js
Initialize a new project with NPM
Install Nodemon globally to automatically reload scripts
Install the Cron NPM package and use it in a script
Look up the cron expression for “every 10 minutes”. Test it and add it ✏️ 0/10 * * * *
Create a Git repo and commit a .gitignore file
Publish the repo on Github. Add the link ✏️ [HERE](https://github.com/rohope26/cron-demo)
Turn it in by posting the Github link in Moodle
